Datos del Problema


Sistema RSA:

    Número primo p = 47

    Número primo q = 61

    n = p × q

    Función de Euler: ϕ(n) = (p - 1) × (q - 1)

    Exponente público e = 17

    Exponente privado d tal que e × d ≡ 1 mod ϕ(n)


Cifrado del mensaje:

    El mensaje original M fue cifrado con RSA:

    C = M mod n

    Luego, se aplicó un cifrado de desplazamiento modular con clave secreta K = 23:

    C′ = (C + K) mod n


Valor cifrado recibido:

    Se obtiene el valor final C′ = 575


Pregunta:

    Recupera el mensaje original M invirtiendo los pasos de cifrado
